# Student REST API

This project is a simple REST API for managing student data.

## Features
- Get all students
- Add a new student
- Update student details
- Delete a student

## Project Structure

API/
api/
rest_main/
students/

## API Endpoints

GET /students  
POST /students  
PUT /students/{id}  
DELETE /students/{id}

## Example Student Data

{
  "id": 1,
  "name": "Rahul",
  "age": 20,
  "course": "Computer Science"
}

## How to Run

1. Activate virtual environment
2. Run the server
3. Open API in browser or Postman
